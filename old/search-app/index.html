DIJKSTRA SEARCH ALGORITHM

HOW IT WORKS

JavaScript

The original application that was made for the coursework was purely a command line application, therefore for a more accessible demo a front end app was written in JavaScript.

Java

The Dijkstra algorithm that the above demo uses is written in Java. This was the primary language that most of my university courses used.

Serverless

Since the original app was just a command line program, it had to be adopted slightly so that it was easier to demonstrate. Therefore the codebase was made to be deployed into an AWS Lambda Function using Serverless.
